## 🚀 Projects I've Contributed To

These are collaborative team projects I worked on as a backend contributor.

### 🏠 Acres — Property Management System
> Python · Flask · CockroachDB · PostgreSQL · pytest

A web-based property management platform built for independent Rwandan landlords to digitise rent tracking, lease management, maintenance tickets, and tenant communication.

**My role: Backend (DSA) Engineer**
- Implemented hash maps for caching and a min-heap for maintenance ticket priority queuing
- Built Flask REST API routes: `/api/v1/property`, `/api/v1/tenant`, `/api/v1/tickets/queues`
- Designed the relational schema and connected the Python backend to CockroachDB
- Wrote pytest test suites for ticket and tenant endpoints
- Enforced PEP8 standards, `.env` config, structured error handling, SSL connectivity, and Git branch strategy

### 🚕 Urban Mobility Data Explorer
> Python · Flask · SQLAlchemy · MySQL · React · TypeScript

An interactive full-stack dashboard for exploring and visualising NYC taxi trip data, with filtering, real-time statistics, and a GeoJSON map of taxi zones.

**My contributions:**
- Built and maintained Flask API endpoints for paginated trip data, aggregated stats, and GeoJSON zones
- Wrote Python data-cleaning scripts to process large datasets before loading into MySQL via SQLAlchemy
- Implemented database models (`Trip`, `Location`) and contributed to query optimisation
- Collaborated via Git feature branches and pull requests across a full-stack team

### 💸 MoMo SMS Database Design & Data Processing
> Python · MySQL · JSON · ERD Design

A relational database system for processing Mobile Money transaction data extracted from SMS/XML records, with structured JSON serialisation for future API consumption.

**My contributions:**
- Parsed and cleaned raw SMS/XML MoMo transaction records using Python
- Contributed to ERD design and reviewed SQL constraints across Transactions, Users, and System_Logs tables
- Designed JSON serialisation schemas mapping relational data to nested API response structures
- Validated test data and supported data dictionary documentation
